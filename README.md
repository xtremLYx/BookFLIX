# BookFlix

A Netflix-style book discovery and library tracking app. Browse, search, and catalog thousands of books with an immersive, cinematic UI. Built with Next.js, Supabase, and the Google Books API.

## Features

- **Book Search** - Search millions of titles via the Google Books API
- **Personal Library** - Organize books in a Kanban-style space
- **Book Clubs** - Join or create reading clubs
- **User Auth** - Sign up and sign in with Supabase Authentication

## How to Run

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- [Next.js 16](https://nextjs.org) - React framework
- [Supabase](https://supabase.com) - Auth and backend
- [Tailwind CSS 4](https://tailwindcss.com) - Styling
- [Framer Motion](https://www.framer.com/motion/) - Animations
- [Google Books API](https://developers.google.com/books) - Book data
